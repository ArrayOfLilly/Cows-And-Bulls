Rules

  • The app generates a secret number with configurable length (3...8).
  • You submit guesses to find the secret within the maximum step count.
  • Bull: correct digit in the correct position.
  • Cow: correct digit in the wrong position.

Scoring

Final score = base score × difficulty multiplier × performance multiplier.

Base score depends on code-space size: log10(combinations) × 100.

Difficulty multipliers

OptionMultiplier
Repeating digits enabled×1.15
Hard mode enabled×1.40
Hide remaining guesses×1.15
Per-guess timer enabledup to ×1.50 (stricter limit = higher multiplier)
Game timer enabledup to ×1.50 (stricter limit = higher multiplier)
Guess pressure×max(1.0, standardGuesses / maxGuesses)

Performance multiplier: min(2.5, standardGuesses / usedGuesses).

First-guess jackpot: if solved in 1 guess, extra + base score × 10.0.

If any enabled timer expires, the game is lost.

Settings

  • Maximum guesses: game ends when this limit is reached.
  • Answer length: length of the generated secret number.
  • Enable repeating: allows repeated digits.
  • Enable hard mode: only latest feedback is shown.
  • Per-guess timer: set a time limit for each guess.
  • Game timer: set a time limit for the whole game.
  • Timer behavior: if any enabled timer expires, the game is lost.
  • Sound: toggle effects and set volume.
  • Theme: choose bull/cow image pair.
  • Language: follow system, English, or Hungarian. Changes may require restart; the app can prompt you to restart.

History and Statistics

  • Use Filter (All/Won/Lost) and Sort (Newest/Oldest/Highest score).
  • Expand a game row to view all guesses and bull/cow results.
  • The Statistics tab shows overview, performance, and most used settings.
